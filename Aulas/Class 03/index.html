Media Query - Padrões

Os breakpoints definem as larguras da tela nas quais devemos ajustar o layout.

Dispostivo Largura da tela (px) Exemplo de media query
Celular pequeno até 480px @media (max-width: 480px) {...}
Celular grande até 768px @media (min-width: 481px) and (max-width: 768px) {...}
Tablet até 1024px @media (min-width: 769px) and (max-width: 1024px) {...}
Desktop acima de 1024px @media (min-width: 1025px) {...}
